This project is a simple AI-based web application that recommends study materials based on the topic entered by the user. The system uses a TF-IDF vectorization technique and cosine similarity to analyze text and find the most relevant resources from a dataset. The backend is built using FastAPI (Python) to handle the recommendation API, while the frontend is developed with Next.js and Tailwind CSS to provide a clean user interface. When a user searches for a topic like “machine learning” or “math”, the application processes the query, compares it with available resources, and returns the most relevant learning materials such as videos or articles. The goal of this project is to demonstrate how a simple machine learning recommendation system can be integrated with a full-stack web application.
